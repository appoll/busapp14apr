Fundamentals

Acesta este un read-only REST API. Datele sunt retrieved prin trimiterea unei cereri HTTP GET la un URL appropriate. Autentificarea nu este necesara, iar operatiile de scriere sunt inexistente.

Tipuri de date supported

Formatul răspunsului este selectat prin adăugarea unei extensii la finalul endpoints descrise mai jos. Aceasta se realizează prin appending unui punct urmat de către unul dintre cuvintele cheie din tabelul următor.

Cuvânt cheie MIME Type Format Observații
json application/json JSON Acesta este formatul de date principal folosit.
js text/javascript JavaScript Allows cross-site requests via the JSON-P convention. Provide a callback query string argument containing the name of your JavaScript callback function.
kml application/vnd.google-earth.kml+xml KML A geographical data format supported only for stops and vehicles. This format can be used to add a "Network Link" to Google Earth. Only the location and a display name of the retrieved items are available at this time.

Data Endpoints

Lista agențiilor

URL: /agencies

Returnează o listă de agenții care pot fi interogate via acest API.

Exemple

  • Listă de agenții în JSON

Proprietățile unei agenții

URL: /agencies/id

Returnează un obiect reprezentând agenția dorită.

Exemple

  • RATP Iași în JSON

Lista rutelor unei agenții

URL: /agencies/id/routes

Returnează o listă a rutelor operated de către agenția selectată.

Exemple

  • Rutele operated de către RATP Iași în JSON

Properietățile unei rute

URL: /agencies/id/routes/id

Returnează un obiect reprezentând ruta dorită.

Exemple

  • Linia 28 aparținând RATP Iași în JSON

Lista stațiilor de pe o rută

URL: /agencies/id/routes/id/stops

Returnează o listă a stațiilor de pe ruta selectată. Această listă reprezintă reuniunea tuturor stațiilor serviced de către toate cursele aparținând acestei rute, și nu este într-o ordine anume. Pentru a obține o listă ordonată a stațiilor este necesară cererea de stații pentru o anumită cursă.

Stațiile pentru cele două direcții ale unei rute bidirecționale sunt often returnate ca și două stații diferite. However, sometimes a terminal of a route may be represented by the same stop in both directions.

Exemple

  • Lista stațiilor aparținând liniei 13 a RATP Iași în JSON

Lista curselor unei rute

URL: /agencies/id/routes/id/trips

Returnează o listă de curse pentru ruta selectată. O cursă descrie o secvență de stații dintre două capete de linie. Majoritatea rutelor au cel puțin două curse și pot avea curse diferite used at specific times of day.

Exemple

  • Lista curselor aparținând liniei 43 a RATP Iași în JSON

Proprietățile unei curse

URL: /agencies/id/routes/id/trips/id

Returnează un obiect reprezentând cursa dorită.

Exemplee

  • O cursă a liniei 6B aparținând Unistil în JSON

Lista stațiilor unei curse

URL: /agencies/id/routes/id/trips/id/stops

Returnează o listă a stațiilor pentru cursa selectată. Stațiile sunt returnate în ordinea sosirii, deci primul și ultimul obiect din listă sunt capetele de linie.

Exemple

  • Lista stațiilor parcurse în timpul unei curse de linia 6B aparținând Unistil în JSON

Propietățile unei stații

URL: /agencies/id/stops/id

Returnează un obiect reprezentând stația dorită.

Exemple

  • Stația Frigorifer în JSON

Lista rutelor care deservesc o anumită stație

URL: /agencies/id/stops/id/routes

Returnează o listă a tuturor rutelor care deservesc stația selectată.

Exemple

  • Rutele care trec prin stația Podu de Piatră în JSON

Lista sosirilor predicted pentru o anumită stație

URL: /agencies/id/stops/id/predictions

Returnează o listă a sosirilor predicted pentru stația selectată, across toate rutele. Predicțiile sunt listate în ordinea sosirii expected.

Exemple

  • Predicted sosiri pentru stația Gară în JSON

Lista sosirilor predicted pentru o anumită rută la o anumită stație

URL: /agencies/id/stops/id/predictions/by-route/id

Returnează o lisă a sosirilor predicted pentru stația selectată, incluzând numai vehiculele de pe ruta specificată. Predicțiile sunt listate în ordinea sosirii expected.

Exemple

  • Predicted sosiri pentru linia 30 la stația Piața Nicolina în JSON
